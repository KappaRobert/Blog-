Как использовать:

Установите зависимости:

bash
pip install flask

Создайте структуру папок:

/your-project-folder
    /static
        /images
    /templates
        index.html
        admin.html
    app.py

Запустите сервер:

bash
python app.py


Основные возможности:

Загрузка изображений (JPG/PNG/GIF)
Сохранение постов в оперативной памяти
Простая админ-панель
Отображение галереи с описаниями
